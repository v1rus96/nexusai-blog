# Google Search Console Setup

Step-by-step guide to verifying TechLion Blog in Google Search Console and submitting the sitemap for indexing.

## 1. Add the Property

1. Go to [Google Search Console](https://search.google.com/search-console).
2. Click **Add property**.
3. Choose **URL prefix** and enter the full site URL (e.g. `https://yourdomain.com`).

## 2. Verify Ownership

### Option A — HTML Meta Tag (Recommended)

1. Google will provide a meta tag like:
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />
   ```
2. Open `src/app/layout.tsx` and add it inside the `metadata` export:
   ```ts
   export const metadata: Metadata = {
     // ...existing fields
     verification: {
       google: "YOUR_VERIFICATION_CODE",
     },
   };
   ```
3. Deploy the change, then click **Verify** in Search Console.

### Option B — DNS TXT Record

1. Copy the TXT record value Google provides.
2. Add a TXT record to your domain's DNS settings with that value.
3. Wait for DNS propagation (can take up to 72 hours, usually minutes).
4. Click **Verify** in Search Console.

## 3. Submit the Sitemap

1. In Search Console, go to **Sitemaps** (left sidebar).
2. Enter `sitemap.xml` in the "Add a new sitemap" field.
3. Click **Submit**.

The sitemap and `robots.txt` are already auto-generated by the Next.js app — no extra configuration needed.

## 4. Expected Indexing Timeline

- **Verification:** Usually instant with the HTML tag method.
- **Sitemap processing:** Google typically processes submitted sitemaps within a few hours to a couple of days.
- **Initial indexing:** Pages usually start appearing in search results within 2–7 days after sitemap submission. Full crawl of all pages can take 1–4 weeks depending on site size and Google's crawl budget.

## 5. Tips

- After submitting the sitemap, use **URL Inspection** to request indexing of high-priority pages immediately.
- Monitor the **Coverage** report for any crawl errors.
- If pages aren't being indexed, check for `noindex` tags or `robots.txt` blocks (the current setup should be clean).
